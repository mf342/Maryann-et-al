This repository contains all code and datasets used to generate the figures and tables in Maryann et al. 2025. In total, the paper includes 10 figures and 5 tables.

For any questions or collaboration inquiries, please feel free to contact me at maryannbenny.fernandes@duke.edu. 

**Data Sources:** All synthetic data were obtained from the NASA JPL Horizons system: https://ssd.jpl.nasa.gov/horizons/

Real-time data were taken with the PROMPT telescopes in Chile during the observations described in the manuscript.

**Licensing:** This repository is released under the MIT License, allowing reuse with attribution.


---------------------------------------------------------------------------------------------------------------------------

Figure 1

Synthetic Horizons data for asteroid 2024 ON (Sept 5–6, 2024, 1-hr cadence, 48 observations, 20 mas noise) and code to reproduce the figure.

Figure 2

Synthetic Horizons data for asteroid 4953 (Oct 30–31, 2024, 1-hr cadence, 48 observations, 20 mas noise) and corresponding code.

Figure 3

Synthetic Horizons data for 20 asteroids over a short observational arc (Sept 5–6, 2024) with code for distance-measurement performance plots.

Figure 4

Synthetic Horizons data for the same 20 asteroids over a multi-night arc (Sept 5–9, 2024). Includes code for reproducing distance-comparison plots.

Figure 5 & Table 1

Real-time PROMPT telescope observations of 2024 ON and 4953, aligned with corresponding Horizons predictions. Includes residual-calculation scripts.

Figure 6 & Table 2

Real-time PROMPT data for 2024 ON (~4 hr arc) with scripts implementing Eq. 1 from the manuscript to compute distance and error.

Figure 7 & Table 3

Real-time PROMPT data for 2024 ON (~7 hr arc) with distance calculations and residual analysis.

Figure 8

Combined synthetic + real-time comparisons for the sample of 20 asteroids. Code for multi-panel visualization included.

Figure 9 & Table 4

Find_Orb-based uncertainty analysis, including input observational files, fitting scripts, and visualization code.

Figure 10 & Table 5

Fractional distance uncertainty calculations and plotting scripts for multi-asteroid comparisons.
